# Personal Page

This is a personal page where I can share my knowledge and experiences with the world.

## How to run

1. Clone the repository
```bash
git clone https://github.com/jorgecatano/jorgecatano.github.git
```

2. ⚠️ Install Jekyll on your local machine if it is not already installed. You can find the installation instructions here: https://jekyllrb.com/docs/installation/

3. Run Jekyll
```bash
bundle exec jekyll serve
```

4. Open the page in your browser using the url http://localhost:3000
